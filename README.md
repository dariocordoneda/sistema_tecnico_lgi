## 🇦🇷 Español

### ¿Qué es?

Sistema de gestión integral para talleres de servicio técnico electrónico. Cubre el ciclo completo de reparaciones: desde la recepción del equipo hasta la entrega al cliente, con control de stock, ganancias y gestión de personal.

---

### ✅ Funcionalidades

| Módulo | Descripción |
|--------|-------------|
| 📋 Fichas de trabajo | Creación, seguimiento y cierre de órdenes de reparación |
| 🔍 Diagnóstico técnico | Registro de fallas con evidencia fotográfica |
| 📦 Control de stock | Inventario de repuestos con alertas |
| 💰 Resumen de ganancias | Filtro por fecha, exportación a Excel y PDF |
| 👥 Gestión de personal | Usuarios con roles y permisos diferenciados |
| 💬 WhatsApp | Notificaciones automáticas al cliente |
| 🗑️ Historial de eliminados | Papelera con trazabilidad completa |
| 📊 Dashboard | Panel con fichas activas y estados en tiempo real |

---

### 🚀 Instalación local

```bash
# Clonar el repositorio
git clone https://github.com/dariocordoneda/sistema_tecnico_lgi.git
cd sistema_tecnico_lgi

# Crear y activar entorno virtual
python3 -m venv venv
source venv/bin/activate

# Instalar dependencias
pip install -r requirements.txt

# Aplicar migraciones
python manage.py migrate

# Crear superusuario
python manage.py createsuperuser

# Correr el servidor
python manage.py runserver
```

> Accedé en **http://localhost:8000**

## 🇺🇸 English

### What is it?

A full-featured management system for electronics repair workshops. Handles the complete repair lifecycle — from device intake to customer delivery — including diagnostics, parts inventory, photo evidence, earnings tracking, and staff management.

---

### ✅ Features

| Module | Description |
|--------|-------------|
| 📋 Work orders | Create, track, and close repair tickets |
| 🔍 Technical diagnostics | Fault logging with photo evidence |
| 📦 Stock control | Parts inventory with alerts |
| 💰 Earnings summary | Date filters, Excel & PDF export |
| 👥 Staff management | Users with differentiated roles and permissions |
| 💬 WhatsApp | Automatic customer notifications |
| 🗑️ Deletion history | Trash with full traceability |
| 📊 Dashboard | Active tickets and real-time status overview |

---

### 🚀 Local Setup

```bash
# Clone the repository
```
